Interface IPromptNumberOptions

Options passed to Prompts.number() or in a session.beginDialog() call to a custom prompt based on the PromptNumber class.

Inherited from IPromptOptions.retryPrompt

  • Defined in botbuilder.d.ts:775

Optional retrySpeak

retrySpeak: TextType

Inherited from IPromptOptions.retrySpeak

  • Defined in botbuilder.d.ts:778

(Optional) SSML to send with the retryPrompt. If the retryPrompt is of type IMessage or IIsMessage, this value will be ignored. If this value is an array a response will be chosen at random.

speak: TextType

Inherited from IPromptOptions.speak

  • Defined in botbuilder.d.ts:769
