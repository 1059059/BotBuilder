Detailed Description

Build a form by specifying messages, fields and confirmations via reflection or programatically.

Template Parameters
TForm state class.

Fields will be defined through reflection over the type T and attributes like DescribeAttribute, NumericAttribute, OptionalAttribute PatternAttribute, PromptAttribute, TermsAttribute and TemplateAttribute. For all of the attributes, resonable defaults will be generated.
