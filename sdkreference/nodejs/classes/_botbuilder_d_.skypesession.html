compareConfidence

  • compareConfidence(language: string, utterance: string, score: number, callback: function): void

  • Inherited from Session.compareConfidence

    • Defined in botbuilder.d.ts:1042

    Lets a dialog compare its confidence that it understood an utterance with it's parent. The callback will return true if the utterance was processed by the parent. This function lets a parent of the dialog handle messages not understood by the dialog.

    Parameters

    • language: string

      The langauge of the utterance taken from IMessage.language.

    • utterance: string

      The users utterance taken from IMessage.text.

    • score: number

      The dialogs confidence level on a scale of 0 to 1.0 that it understood the users intent.

    • callback: function

      Function to invoke with the result of the comparison.

        • (handled: boolean): void

        • Parameters

          • handled: boolean

            If true the dialog should not process the utterance.

          Returns void

    Returns void

endDialog

  • endDialog(result: string | IMessage | IDialogResult<any>, ...args: any[]): Session

  • Inherited from Session.endDialog

    • Defined in botbuilder.d.ts:1030

    Ends the current dialog and optionally sends a message to the user. It's typically more efficient to call endDialog() with a message then it is to call send() seperately before ending the dialog.

    If a message is sent to the user it will be sent before the dialogs parent is resumed. The parent will be resumed with an IDialogResult.resumed reason of completed.

    Parameters

    • result: string | IMessage | IDialogResult<any>
      • result: {string} - Text of a message to send the user. The message will be localized using the sessions configured localizer. If arguments are passed in the message will be formatted using sprintf-js.
      • result: {IMessage} - Message to send the user.
      • result: {IDialogResult} - Optional results to pass to the parent. If endDialog() is called without any arguments the parent will be resumed with an IDialogResult.resumed reason of completed.
    • Rest ...args: any[]

      Optional arguments used to format the final output text when result is a {string}.

    Returns Session

  • ngettext(msgid: string, msgid_plural: string, count: number): string

  • Inherited from Session.ngettext

    • Defined in botbuilder.d.ts:970

    Loads the plural form of a localized string for the messages language. The output string will be formatted to include the count by replacing %d in the string with the count.

    Parameters

    • msgid: string

      Singular form of the string to use as a key in the localized string table. Use %d to specify where the count should go.

    • msgid_plural: string

      Plural form of the string to use as a key in the localized string table. Use %d to specify where the count should go.

    • count: number

      Count to use when determining whether the singular or plural form of the string should be used.

    Returns string
