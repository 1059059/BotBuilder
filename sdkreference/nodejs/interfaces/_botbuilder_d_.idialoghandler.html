Interface IDialogHandler

Signature for function passed as a dialog handler.

WARNING: The dialog handler will be called a second time if the initial call invokes one of the built-in Prompts or session.beginDialog(). Your handler should guard against situations which could result in starting a dialog a second time:

example

var bot = new builder.BotConnectorBot();
bot.add('/', function (session, results) {
    if (results && results.resumed) {
        // Second call to handler returning results from prompt.
        session.send("Hello %s.", results.response);
    } else {
        // First call to handler.
        builder.Prompts.text(session, "Hi! What's your name?");
    }
});
