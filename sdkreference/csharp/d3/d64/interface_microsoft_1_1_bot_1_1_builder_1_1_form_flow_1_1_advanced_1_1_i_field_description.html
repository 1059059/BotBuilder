Detailed Description

Describe the information displayed about a field and its values.

Throughout this class Description refers to the name of a field or a value whereas "terms" tell what people can type to match the field or terms in it. When generating terms it is a good idea to include anything that might be reasonable for someone to type. The form dialog itself will help clarify any ambiguity. One way to do this is to use TermsAttribute.MatchPhrase which ensures that Language.GenerateTerms is called on your base terms.
